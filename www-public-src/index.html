Работа из консоли

Основные команды

  • var x = createTortoise() — создать черепаху в начале координат.
  • var t = createTortoise(100, 50, "green") — создать черепаху в нужном месте определённого цвета
  • t.go(50) — пройти вперёд на нужное расстояние
  • t.rotate(45) — повернуться против часовой стрелки на нужное количество градусов
  • t.tailDown() — опустить хвост, чтобы рисовать линию.
  • t.tailUp() — поднять хвост, чтобы просто перемещаться.
  • t.setColor("red") — поменять цвет черепахе.

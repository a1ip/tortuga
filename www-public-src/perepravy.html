Работа из консоли

Основные команды

  • help() — отобразить список доступных команд
  • lessons() — отобразить список уроков
  • lesson(0) — выбрать нужный урок
  • problems() — отобразить список задач
  • start(4) — выбрать нужную задачу
  • show() — прочитать условие задачи
  • move("волк, лодочник") — перевезти героев с берега на берег.
